Ubuntu部署hexo

发表于 2022-10-26 更新于 2022-10-27 分类于 hexo
在Windows的Linux子系统Ubuntu上部署Hexo个人博客

Ubuntu安装hexo

1、从NodeSource安装node.js

1.1 下载并执行NodeSource脚本

curl -sL https://deb.nodesource.com/setup_14.x | sudo -E bash -

1.2 安装node.js和npm

sudo apt install nodejs

1.3 验证安装

node --version
npm --version

2、部署hexo

2.1 下载hexo

sudo npm install hexo-cli -g

2.2 初始化

hexo init

2.3 安装其它可能需要的组件

npm install

2、Git配置

2.1 配置user.name和user.email

git config --global user.name "HN-YZH"  
git config --global user.emal "*176*****693@163.com"

2.2 生成ssh公钥和私钥

ssh -keygen -t rsa -C "176*****693@163.com"

2.3 查看生成的公钥

cd ~/.ssh 
ls 
cat id_rsa.pub

2.4 验证Git

 ssh -T git@GitHub.com

3、hexo关联Git

3.1 安装hexo-deployer-git组件

npm install --save hexo-deployer-git

3.2 修改hexo根目录下的_config.yml

deploy:  
  type: git  
  repository: git@github.com:HN-YZH/HN-YZH.github.io.git    
  branch: master

4、hexo更换主题next

4.1 下载next主题

git clone https://github.com/iissnan/hexo-theme-next themes/next

4.2 使用next主题

//修改hexo _config.yml
# Extensions
## Plugins: https://hexo.io/plugins/
## Themes: https://hexo.io/themes/
## theme: landscape
theme: next

4.3 next 添加本地搜索 local Search

//安装 hexo-generator-searchdb
npm hexo-generator-searchdb --save
//修改 hexo 配置文件_config.yml 新增以下内容
# local search
search:
  path: search.xml
  field: post
  format: html
  limit: 10000
//修改 next 配置文件 _config.yml
# Local search
local_search:
  enable: true

5、首页文章显示简介

//修改 默认文章模板 /scaffolds/post.md
title: {{ title }}
date: {{ date }}
tags:
categories:
description: 点击阅读前文前, 首页能看到的文章的简短描述

6、菜单图标和数量的气泡显示

# Enable / Disable menu icons / item badges.
menu_settings:
icons: true
badges: true

7、设置代码高光显示

  light: vs2015
  dark: vs2015
